# Fran_Game
Unity와 C#스크립트, Google Cardboard SDK를 이용해 만든 안드로이드 VR 게임입니다.
Asset을 수정해 그림이 놓인 아파트 환경을 3D환경에서 구현했습니다.
화면을 클릭해 특정 Boundary 안 무작위 위치에 생성되는 오브젝트를 바라보고 클릭하면 소리를 재생하고 다시 무작위 위치에서 재생성됩니다.
